# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-07-26

### Added
- Functional and styled Contact Form on the Contact page.
- Enhanced single post page layout with a new two-column design.
- **Recent Posts Sidebar** on the single post page to improve content discovery.
- **"You Might Also Like" Section** at the bottom of posts to recommend similar articles based on tags.
- **Social Sharing Buttons** (Twitter, Facebook, LinkedIn, Copy Link) on each post.
- Attribution link in the footer, suitable for a sellable template.
- Expanded content for all sample blog posts to provide a more realistic demo.

## [1.0.0] - 2024-07-25

### Added
- Initial release of the IgnitePress blog template.
- Fully responsive design with a mobile-first approach.
- Light and Dark mode with theme persistence via local storage.
- Dynamic landing page to welcome visitors.
- "All Articles" page with live search, tag filtering, and pagination.
- Individual blog post pages and dynamic category pages.
- SEO enhancements with dynamic page titles and meta descriptions.
- Pre-built utility pages: About, Contact, Start Here, Privacy, and Resources.
- Core components: Sticky Header, Multi-column Footer, and a Scroll-to-Top button.
- All content is easily managed from a single file (`src/data/posts.ts`).
